## Clonar este proyecto

```bash
> https://github.com/ThonyDroidYT/YukinikoBot
```

```bash
>Command BOT $
```

## Instalar las dependencias:
Antes de ejecutar el siguiente comando, asegúrese de estar en el directorio del proyecto que acabas de clonar !!

```bash
> cd YukinikoBot
> bash install.sh
```

### Iniciar Bot
```bash
> npm start
```
#### Para Ver Comandos
> .menu
```
